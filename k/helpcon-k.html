명령어 Help: 대화
기본적으로 모든 대화는 채팅 입력창에서 이루어집니다.

말하기:

키보드로 말하고자 하는 말을 입력합니다. 화면에 보이는 사용자에게만 말이 전달됩니다.

귓속말:

원하는 사람에게만 들리게 말할 수 있습니다. 사용 방법은 "를 먼저 써준 후에 사용자 ID를 입력하고 한칸 띄고 말하고자 하는 말을 쓰면 됩니다..

귓속말은 채팅 창에 분홍색으로 나옵니다. 다른 사람의 귓말을 듣지 않으려면 /귓말 끔이라고 하면 됩니다. 다시 켜려면 /귓말 켬을 입력합니다.

채팅:

게임 화면에 말 풍선이 보이지않고, 화면 하단의 전체 채팅창에서만 보이도록 채팅을 할 수 있습니다. 채팅은 모든 사용자에게 전달됩니다. 채팅은 채팅 기능을 끄면 보이지 않습니다. 사용 방법은 &[말하고자 하는 말]

채팅은 채팅 창에 하늘색으로 표시됩니다. 채팅을 끄려면 /채팅 끔이라고 입력합니다. 다시 켜려면 /채팅 켬이라고 입력합니다.

혈맹 채팅:

같은 혈맹원간에만 보이도록 채팅을 할 수 있습니다. 사용 방법은 @[말하고자 하는 말]입니다.

혈맹 채팅은 채팅 창에 연녹색으로 표시됩니다.

상위 혈맹 채팅:

혈맹 연합의 경우 상위 혈맹 채팅을 사용할 수 있습니다. 사용 방법은 %[말하고자 하는 말]입니다.

상위 혈맹 채팅은 채팅 창에 진한 분홍색으로 표시됩니다.

파티 채팅:

같은 파티원간에만 보이도록 채팅을 할 수도 있습니다. 사용 방법은 #[말하고자 하는 말]

파티 채팅은 채팅 창에 연보라색으로 표시됩니다.

채팅 기능 On/Off:

채팅창에서 전체 채팅을 보고 싶지 않을 때에는 채팅 입력창에서 '/채팅 끔' 라고 입력합니다. 다시 채팅을 보고 싶을 때에는 '/채팅 켬' 이라고 입력합니다.

특정 사용자 차단:

채팅창에서 특정인으로부터의 메시지만을 제외하고 싶을 때에는 채팅 입력창에서 '/차단 [유저 ID]' 을 입력합니다. 특정 캐릭터를 차단하면 그 사실이 그 캐릭터에게도 알려집니다. [유저 ID]를 입력하지 않으면 현재 차단하고 있는 캐릭터의 목록을 보여줍니다. 현재 차단하고 있는 사용자로부터의 메시지를 다시 받고 싶으면 다시 한번 '/차단 [유저 ID]'를 입력하면 됩니다.
이 기능은 채팅 윈도우에 있는 차단하기를 원하는 유저의 이름을 오른쪽 버튼으로 더블 클릭해도 동작합니다.

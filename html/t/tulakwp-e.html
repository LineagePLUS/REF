Blacksmith Tulak:

[Setting a storage house password]
 1. Click "Set (change) storage house password."
2. Click "OK" while password input window is blank.
3. Enter a six-digit password and click "OK".
4. Enter the password again and click "OK".
5. The password is set.

[Using a storage house]
 1. Select "Find item".
2. Enter the storage house password and click "OK".
3. You may place items in or take items from the storage house.
4. The storage house password also accesses the Blood Pledge storage house.

[Changing a storage house password]
 1. Click "Set (change) storage house password."
2. Enter the current password and click "OK".
3. Enter a new six-digit password and click "OK".
4. Enter the new password again and click "OK".
5. The password has been changed.

[Removing a storage house password]
 1. Click "Set (change) warehouse password."
2. Enter the current password and click "OK".
3. Leave the password input window blank and click "OK".
4. Click "OK" again.
5. The storage house password has been removed.

[Special notes]
 1. The storage house password must contain six digits.
2. The password may be set in game options.

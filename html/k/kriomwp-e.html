Kriom the Dwarf:

Setting the Storage House Password
 1. Click "Set/Change Password".
2. Click "OK".
3. Enter a new password and click "OK". (Must be a six-digit number.)
4. Enter the new password one more time and click "OK" in order to confirm the new password.
5. The storage house password is now set.

Using the Storage House
 1. Click "Retrieve Item".
2. Enter the six-digit password that you set and click "OK".
3. Get the item(s) that you need from the storage house.
4. You can also use your storage house password when getting items from the Blood Pledge storage house.

Changing the Storage House Password
 1. Click "Set/Change Password".
2. Enter the current password and click "OK".
3. Enter a new password and click "OK". (Must be a six-digit number.)
4. Enter the new password one more time and click "OK" in order to confirm the new password.
5. The storage house password is now changed.

Removing the Storage House Password
 1. Click "Set/Change Password".
2. Enter the current password and click "OK".
3. Click "OK" without entering anything into the input window for changing the password.
4. Click "OK" without entering anything into the input window for confirming the new password.
5. The storage house password is now removed.

Special Note
 1. The storage house password must be a six-digit number.
2. You can set the storage house password at Game Options.

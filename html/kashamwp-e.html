Dwarf Kasham:

Setting a Storage House Password
 1. Click "Set/Change Storage House Password".
2. Click "OK" without entering anything into the input window.
3. Enter a new password and click "OK". (Six digits required.)
4. Enter the new password again and click "OK" to confirm.
5. The storage house password is set.

Using a Storage House
 1. Select "Get Item".
2. Enter the six-digit password that you set and click "OK".
3. Get the item(s) that you need from the storage house.
4. Storage house password also works on your Blood Pledge storage house.

Changing a Storage House Password
 1. Click "Set/Change Storage House Password".
2. Enter the current password and click "OK".
3. Enter a new password and click OK. (Six digits required.)
4. Enter the new password again and click "OK" to confirm.
5. The storage house password is changed.

Removing the Storage House Password
 1. Click "Set/Change Storage House Password."
2. Enter the current password and click "OK".
3. Click "OK" without entering anything into the input window.
4. Click "OK" without entering anything into the input window to confirm.
5. The storage house password is removed.

Special Notes
 1. The storage house password must be a six-digit number.
2. You can set the storage house password at Game Options.

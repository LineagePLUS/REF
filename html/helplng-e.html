Command Help: Language
Lineage currently supports English, Korean, Japanese, and Chinese.

Toggle English-Korean:

To toggle English-Korean input mode, press the Space key while pressing the Shift key. The icon represents English input mode and represents Korean.

Korean Input Methods

There are two Korean input methods. 3 set users may type '/3set' from the Chat Window to use 3 set keyboard. Switch to 2 set by typing '/2set'.

Toggle to another language version

Lineage sets its language setting to the same language that Windows is set up to use (for instance, if you use the English version of Windows95, then Lineage will display all system messages in English). But if you want to change this default language setting, type '/lang korean' ,'/lang japanese' or '/lang english' to change the game's language setting to Korean, Japanese or English, respectively.

Command Help: Items
Picking Up Items:

If you wish to pick up an item which has fallen on the ground, click the item or use the mouse to drag the item to your character.

If two or more items overlap in one location, press the right mouse button to select the next item.

Items which are grouped together in one bundle, such as adena and arrows, can be picked up together with one click. You can type '/getall' in the Chat Window or you can toggle Get All: On/Off under Game Options.

Using Items:

If you want to use an item which you have, double-click the respective item in your Item Window.

Discarding Items:

If you want to discard an item which you have, drag the respective item from the Item Window to the ground and release the mouse button.

Organizing Items:

Drag an item to the location on the Item Window where you want it to be.

Selecting More Than One Item:

To select more than one item from the Item Window, just click the items while holding down the Shift key. You can use this method to discard many items at once.

Exchanging Items:

If you want to exchange items with another character safely, type "/exchange" in the chatting window while looking at each other. If you input this command, the Item Exchange Window will appear at the left side of the game screen. Drag the item or items you wish to give the other person from your Item Window to the top of the Exchange Window. The items which the other person is giving you will appear at the bottom of your Exchange Window. After verifying the item or items, press the OK icon and the transaction is completed. If you click the Cancel icon at any time during the transaction or leave the game, the transaction is cancelled.

Selling Items:

You can open a personal shop and sell items to other characters. To do this, type "/shop" in the Chat Window. If you input this command, a Shop Window appears on the left side of the screen. Drag the items to sell from your Item Window to the Shop Window. And input the sale prices and quantities of the items. Finally, after checking the items to sell, click the "OK" icon and switch to shop mode. Characters in shop mode can sell things in the same way as merchant NPCs. To get out of shop mode, input "/shop" again in the Chat Window.

Using the Item Shortcut Keys:

With the F5 - F12 keys, you can conveniently use items which you utilize often without having to open the Item Window each time. Default settings are red potions for F5, orange potions for F6, clear potions for F7, and cyan potions for F8. After opening the Item Window, you can change the assignments by dragging each item to a slot in the Shortcut Window at the bottom right of the screen. For example, if you want to designate F5 for using the Wand of Call Lightning, just open the Item Window and drag the Wand of Call Lightning that is there to the F5 button.
